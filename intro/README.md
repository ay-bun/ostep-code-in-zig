
# Intro
## General Steps 

   1. To make a binary and run, use these commands :

>
>   make cpu
>   ./cpu <any-string-you-like>

   2. To stop, hit CTRL + C.

   3. To clear files generated :

>
>   make clean
>

   4. That's it.

# cpu.zig
   For every second, it will print the string you enter. To run,

>
>   ./cpu <string>
